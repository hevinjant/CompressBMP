Description:
A command-line terminal program that compresses and decompresses a BMP image file.

How to run the program:
- open command line terminal, and go to the project directory.
- run 'g++ task1.cpp'.
- run './a.out [flag] [filename] [password]'.

Instructions:
- a password is a must, and should be of 6 characters.
- set the flag to 'g' for gray scale or 'c' for color.
